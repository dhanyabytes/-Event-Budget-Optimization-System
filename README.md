# Event-Budget-Optimization-System


## What It Does

This project is about making event budgets better using a way of making decisions with artificial intelligence. It helps people put their money in the places like the venue, food, advertising and getting things from one place to another.

## The Problem We Are Trying To Solve

When people plan events they often do not use their budgets well because they do not plan ahead. This means they might spend much money or not use what they have wisely.

## How We Solve The Problem

The event budget planner needs to know some things from you such as:

- How money you have for the event

- What kind of event you are planning

- What is most important, to you

Then it tells you how to split your money in a way that makes sense.

## Tools We Use

- Python programming language

- A basic artificial intelligence approach that uses logic

## Getting Started

1. Make sure you have Python on your computer (if you do not have it you need to install it)

2. Get a copy of our event budget planner code
